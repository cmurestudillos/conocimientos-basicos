# Plantilla - Elementos basicos de React
Plantilla básica que consta de los elementos esenciales para comenzar a crear una aplicación React.

En este proyecto podras ver:
* Una estructura de diseño de proyecto tipica
* Configuración de babel.
* Instalación y configuración del webpack.
* Eslint configuración.
* Configuración de SCSS.
* Principales componentes React.

## Desarrollado con:
* [Node.js](https://nodejs.org/en/)
* [React](https://reactjs.org/)
* [Babel](https://babeljs.io/)
* [Webpack](https://webpack.js.org/)
* [SCSS](http://sass-lang.com/)

A continuacion expongo uan serie de pasos que permitiran en funcionamiento del proyecto en el ordenador para fines de desarrollo y prueba.

### Requisitos
* Node 8.x
* Npm 3.x
* Instalar node modules dentro de la carpeta del proyecto (cd nombre-proyecto)
   - npm install

### Construir la aplicacion
   - npm run build (Contruir la aplicacion)
   - npm run build:watch (Contruir la aplicacion y visionar los diferentes cambios)
   - npm run build:dev
   - npm run build:prod

### Ejecutar aplicacion
 - npm start (Ejecutarla en terminal)
 - npm run serve:dev (Ejecutarla en servidor de desarrollo)
 - npm run serve:dev:dashboard (Ejecutarla en servidor de desarrollo y panel de control)
 - npm run serve (Ejecutarla en vivo)
 - npm run serve:prod (Ejecuta la aplicación utilizando la configuración de producción e iniciará la aplicación utilizando _live-server_)
 
 Puedes visualizar y aprender mas en Documentacion en la web oficial de [React](https://reactjs.org/).
